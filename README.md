## Installation

Requires:
- [React.js](https://react.org/)  to run.

Install the dependencies and devDependencies and start the server.

```sh
cd Ferrari_Page
npm start
```

If want run in production ...

```sh
cd Ferrari_Page
npm i
npm run build
serve -g build
```
